# 확장 실행 방법

## 1. git clone https://github.com/yang-dongmin/NECO.git

## 2. npm install (의존성 설치)

## 3. npm run compile (빌드)

## 4. src/extension.ts에서 F5 -> VSCode 확장 개발 or VScode extension development

## 5. 원하는 함수, 영역 드래그

## 6. Ctrl + Shift + P or Cmd + Shift + P

## 7. Neco 검색 후 실행

## 8. 주석 생성
